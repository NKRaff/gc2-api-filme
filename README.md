# NKRaff-gc2-api-filme
API desenvolvida como uma atividade de Gestão de Configuração II

---

## Instruções para executar projeto

### 1. Instalar dependências
Execute o comando abaixo para instalar as dependências do projeto:
`npm install`

### 2. Configurar variáveis de ambiente
Crie uma cópia do arquivo `.env.example`, renomeie para `.env` e configure a porta do servidor nesse arquivo.

### 3. Executar o projeto
No terminal, execute:
`npm run start`
